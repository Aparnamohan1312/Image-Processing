Introduction:

A region of interest (ROI) is a portion of an image that you want to filter or perform some other operation on. ROI is done by creating a binary mask, which is a binary image that is the same size as the image we want to process with pixels that define the ROI set to 1 and all other pixels set to 0. The concept of a ROI is commonly used in many application areas, for example, in medical imaging. Here I hace used this significant ROI technique to detect a particular portion of the input image liek the medical seal in the certificates for forgery detection.

Mathematical Interpretation:

Spatial scale is a scaling parameter for resizing the proposal according to the feature map dimensions. Let's say in our network, the image size is 1056x640 and due to many convolution and pooling operations, the feature map size reduced to 66x40, which is being used by ROI pooling. Now the proposals are generated based on input image size, so we need to rescale the proposals to feature map size. In this case, we can divide all dimensions of proposal by 16 (1056/66=16 or 640/40=16). So the spatial scale will be 1/16 in the DEMO depicted below.

Proposed Methodology

Many common image operations are performed using Region of Interest in OpenCV. A ROI allows us to operate on a rectangular subset of the image.The typical series of steps to use ROI is: create a ROI on the image, perform the operation you want on this subregion of the image, reset back the ROI. The proposed work includes detection a rectangular or square portion of a coloured strip segment in the input images by applying several techniques like masking,blurring,drawing contours and implementing certain mathematical formula like the distance formula to control the detected portion of the image and restricted it to certain values of pixels in the cartesian x and y coordinates. The HSV, LAB values of the detected ROI portion is then obtained as a result. These values are stored as csv files for further processing which can be used as input dataset for predictions using Machine Learning.
