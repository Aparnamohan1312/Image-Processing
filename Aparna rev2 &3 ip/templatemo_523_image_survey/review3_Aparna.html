WHAT IS HOUGH SPACE?

When we work with images, we can imagine the image being a 2d matrix over some x and y coordinates, under which a line could be described as y = mx + b. But in parameter space, which we will call Hough space, that same line can be represented as m vs b instead, so the characterization of a line on image space will be a single point at the position m-b in Hough space.
